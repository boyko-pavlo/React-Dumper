# React Dumper

It is simple React Dumper.



## Installation:
add dependency

```bash
npm install git+ssh://git@github.com/boyko-pavlo/React-Dumper.git
```

or

```bash
yarn add git+ssh://git@github.com/boyko-pavlo/React-Dumper.git
```

## Usage :

Add `Dumper` to your component:

```tsx
  <Dumper data={{ some: 'data'}} />

```
